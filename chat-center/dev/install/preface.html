Preface

This document explains how to install the Chat Center module.

This is not an actual installation guide. It is intended only to demonstrate some features of AscciiDoc, Asciidoctor, and Antora.

Audience

This guide is primarily intended for implementation personnel responsible for performing the initial installation of the Chat Center module, or tasked with performing an update. Secondary audiences include system administrators, security administrators, and database administrators.
