Starting the user console

After the relevant database details have been configured, the user console can be started.

  1. Access the UI folder.

  2. To start the console, perform the action for the relevant operating system being used:

    • Windows — double-click the ui-con.bat file

    • Unix — execute the following command: java –jar uicon.jar

  3. Open the console.

Appending –pws <host:port> (or –pws <IP address:port>) to the Unix command enables it to be run as a web service.
